3-Step Search Tool

Visitors choose Degree (Bachelor's, Master's, PhD, Associate), Language (English / Turkish / Arabic), and Major, then click Search to find matching programs.

Animated Stats

Numbers count up from 0 when the section scrolls into view. 500+ admissions, 10+ years, 70+ advisors, 60+ universities — builds trust at first glance.

How to Use

Select options from each dropdown then click the orange "Search" button to filter available programs and universities that match your criteria.

User Flow — Find a Program

  1. Select Degree type from the first dropdown (e.g., Bachelor's)
  2. Pick the preferred Language of instruction (English / Turkish / Arabic)
  3. Choose your desired Major (Medicine, Engineering, Business, etc.)
  4. Click the orange Search button — results appear below
